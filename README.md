## What AIUI provides

AIUI is aimed at developers building with the Rokid AIUI framework and AI glasses experiences. It gathers the essentials needed to bootstrap projects quickly, review reference implementations, and keep assistant instructions organized in a clean, structured form.

This repository is a practical fit when you want a repeatable starting point for AIUI agent work. Between the project scaffolding, sample assets, and Chinese/English documentation, it reduces setup overhead and makes the path from idea to implementation easier to follow.

A common workflow is:

1. Read through the documentation to learn the AIUI project layout.
2. Use the CLI to scaffold a new agent project.
3. Inspect the generated files, including app.js, app.json, AGENTS.md, and index.ink.
4. Modify the sample app or feature demo to suit your scenario.
5. Expand or adjust the skill files for your coding assistant workflow.
